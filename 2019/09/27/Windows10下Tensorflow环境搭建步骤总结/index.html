Windows10下Tensorflow环境搭建步骤总结

2019-09-27
教程, 深度学习

TensorFlow是一个基于数据流编程（dataflow programming）的符号数学系统，被广泛应用于各类机器学习（machine learning）算法的编程实现，其前身是谷歌的神经网络算法库DistBelief。搭环境永远是最容易出错的事情，这学期选了《Python程序设计——深度学习及应用》这门课，所以借这个机会重新把安装流程梳理一下，记录一下自己容易犯错的地方。

Anaconda的安装

  1. 进入conda的主页 https://www.anaconda.com/
  2. 选择操作系统，点击download下载
  3. 下载完成后，双击打开，选择路径，开始安装，中途选择添加环境变量和选择为默认python版本（如果先前已经有安装python，可以不选）
  4. 安装完成后，cmd命令行输入conda，如果出现如图所示，则安装成功，否则须手动设置环境变量

Tensorflow的安装

  1. Win+Q后输入Anaconda Navigator并打开

  2. 点击左侧environments，后点击Create

  3. 自己起一个环境命名，点击Create

  4. 环境完成后点击环境右方的按钮，选择open in terminal

  5. 打开终端后，输入

  6. 这里安装完成后，CPU版就已完成，若安装GPU版tensorflow，请检查是否含有以下两个文件，若没有请搜索这两个文件并安装搜索方式为在上方的搜索栏里输入两个包名然后安装即可。

  7. 该部分完成后进入anaconda的目录，进入envs文件夹，该文件夹管理着所有安装的环境。

  8. 设置环境变量，控制面板——>系统和安全——>系统——>高级系统设置。点击环境变量后，编辑系统变量中的path变量。将环境及环境下属的scripts目录复制到此处，至此环境安装完成。

  9. 打开cmd，输入py,进入python命令行，输入import tensorflow，如果不报错，则安装成功。

Pycharm的安装

  1. 进入官网 http://www.jetbrains.com/pycharm/?fromMenu
  2. 选择对应的操作系统，下载
  3. 安装
  4. 配置环境变量

在Pycharm中导入tensorflow

  1. 在pycharm中新建一个project
  2. 点击File——>Settings，进入设置页面,选择Project Interpreter
  3. 点击解释器路径旁的设置图标，选择add
  4. 点击Conda Environment，选择Existing environment，将路径修改为tensorflow下的python.exe，切记不要选成原始的python路径！那样会无法找到tensorflow！
  5. 修改完成，一起来运行一波tensorflow代码试试看吧！
